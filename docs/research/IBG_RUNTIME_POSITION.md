# IBG Runtime Position

**Status:** Research Wave 1A — architectural clarification (documentary only)  
**Authority:** None — operational behavior is defined in `docs/governance/IBG_*` and `services/api/app/instrument_geometry/body/ibg/`

---

## Position

```text
IBG is not a semantic research sandbox.

IBG is the governed semantic runtime bridge
between extraction cognition
and operational geometry workflows.
```

**Sandbox** (`vectorizer-sandbox`) may discover semantics, evaluate modes, and hold archaeology.  
**IBG** operationalizes **provenance-bearing, human-reviewable body evidence** under intake gates and review packages — it does not grant autonomous semantic authority.

---

## Runtime boundary (do not collapse)

| Layer | Role |
|-------|------|
| Extraction / vectorizer spine | Produce governed artifacts (DXF, contours, modes) |
| Cognition lab (external repo) | Experiments, evaluation, incubation |
| **IBG runtime** | Bridge evidence → candidates → gate → review |
| Governance | Ratifies what may run in production defaults |
