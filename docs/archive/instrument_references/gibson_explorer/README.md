## Purpose

specifications used in the Smart Guitar design. It was used to correct
and confirm values in:

## Smart Guitar Differences (intentional)

|---|---|---|---|
| Fret count | 22 | 24 | Extended range |
| Neck joint | Set neck | Bolt-on | Serviceability |
| Tuning | Traditional headstock | Headless | Design intent |
