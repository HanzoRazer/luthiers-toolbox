# Gibson EDS-1275 Double Neck - Reference Source

**Source:** Guitar Plans archive
**Date added:** 2026-03-29

## Overview

The EDS-1275 is a double-neck variant of the Gibson SG body.
- Lower neck: standard 6-string SG (628.65mm scale)
- Upper neck: 12-string SG (628.65mm scale)
- Body: SG double-cutaway extended for two neck pockets
- Hardware: two independent bridge/tailpiece sets

Body join fret and neck specs identical to SG Standard on both necks.
SG Standard spec: `../gibson_sg/` and
`services/api/app/instrument_geometry/specs/gibson_sg.json`

No separate spec file - use SG Standard spec for neck geometry.
Future use: multi-neck body geometry in body outline engine.
