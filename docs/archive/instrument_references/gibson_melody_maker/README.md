## Verified Specifications

### Melody Maker '59
| Specification | Value |
|---|---|
| Scale length | 628.65mm (24.75") |
| Nut width | 43.27mm (1.703") |
| End of board width | 57.56mm (2.266") |
| Neck thickness 1st fret | 22.86mm (0.900") — neck only |
| Neck thickness 12th fret | 24.638mm (0.970") — neck only |
| Headstock angle | 17° |
| Joint angle | 1.5° — shallowest in Gibson library |

### Melody Maker D '65
| Specification | Value |
|---|---|
| Scale length | 628.65mm (24.75") |
| Nut width | 42.85mm (1.687") |
| End of board width | 57.56mm (2.266") |
| Neck thickness 1st fret | 20.32mm (0.800") — same as Explorer |
| Neck thickness 12th fret | 25.4mm (1.000") — thickest 12th in library |
| Headstock angle | 17° |
| Joint angle | 2.5° |

---

## Notable Characteristics

- Both variants have the shallowest joint angles in the Gibson spec library
- The '65 D has the most extreme neck taper: 5.08mm growth vs Explorer's 1.27mm
- Student/budget market instrument — shallow angles simplified manufacturing
