The Core App Concept

What you're describing is essentially a rosette tile planner — a radially-symmetric drag-and-drop canvas where:

  • The wheel is the background, showing concentric ring zones (outer binding, purfling channels, soundhole ring)
  • The tile library contains fundamental motif shapes pre-fitted to the arc geometry — herringbone strips, checkerboard blocks, Celtic knot segments, solid color fills, abalone pieces
  • Radial symmetry is the killer feature — drag one tile into one segment and it auto-mirrors across all 12 positions (or 6, 8, 16 depending on the pattern count)
  • Users can mix zones — herringbone on the outer ring, checkerboard on the inner, geometric on the middle

Technically Interesting Challenges

The tricky part is that these aren't rectangular tiles — they're arc-fitted trapezoids that change proportion based on which ring they're in. An outer-ring piece is wider than an inner-ring piece for the same angular span. Your generator would need to parametrically reshape tiles as they're dropped into different radial zones.

What Would Make This Sing

  • Real dimension readout (in inches) so the layout is actually buildable
  • Export to SVG/DXF for CNC toolpath handoff
  • A "symmetry mode" toggle (bilateral vs rotational vs full)
  • Snap-to-ring behavior when dragging

This fits naturally into the Production Shop platform — it could live alongside your existing CAM toolpath tools and feed directly into a cut file. Want me to build a working prototype of this canvas?

What the Manufacturing Intelligence layer adds:

Header warning badge — always visible regardless of which tab you're on. Pulses red if there are errors, static amber for warnings. Click it from anywhere to jump directly to the Mfg tab.

Manufacturability Score (0–100) — computed live every time you place a tile. Starts at 100, deducts 20 per error, 8 per warning, 2 per info flag. Color-coded green / amber / red with a progress bar.

6 check categories, all geometry-exact:

  1. Fragile arc (< 0.060") — ERROR. Pieces that will crack during cutting. Shows exactly which cells, with ring/segment coordinates.
  2. Narrow arc (0.060"–0.100") — WARNING. Possible but requires care.
  3. Shallow ring depth (< 0.060") — ERROR. Channel too thin to hold inlay reliably.
  4. Extension tab arc — WARNING when tab width goes below 0.050" at current segment count.
  5. Empty purfling — INFO. Main channel filled but adjacent purfling ring is bare — probably unintentional.
  6. Symmetry mismatch — WARNING when quadrant ×4 symmetry is set on a segment count not divisible by 4.

Warning overlays on the wheel — flagged cells get red crosshatch (errors) or amber tint (warnings) with a colored center pip. Toggle on/off without leaving the Mfg tab. Clicking a cell tag in any flag card flashes that cell yellow on the wheel and jumps back to the tile view.

Auto-clear — error-level flags with fixable geometry (fragile arc, shallow ring) offer a one-click "Auto-clear flagged cells" that removes the problematic fill and reruns all checks immediately.
