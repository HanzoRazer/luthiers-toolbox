Architecture
Complete pipeline:
1. contourToSpline(pts) — Catmull-Rom through body outline
2. sampleSpline(spline, n) — equidistant points along path
3. normalVector(pts, i) — perpendicular at each sample
4. placeVineSegment(pt, n, len, p) — vine tile at each point
5. offsetStrip(path, w) — CNC inner/outer offset
6. SVG/OpenSCAD export with kerf compensation
