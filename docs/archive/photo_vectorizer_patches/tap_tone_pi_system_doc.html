06The Three-Phase Measurement Loop

The central problem in lutherie plate work is that free-plate measurements (taken before assembly) do not predict installed-plate behavior (after the box is closed). The three-phase loop is designed to characterize both states and quantify the gap between them — making wolf notes a pre-assembly calculation rather than a post-assembly crisis.

PHASE 1 ─ Free Plate (before assembly) │ │ Plate on foam · suspended at nodal intersection │ Single mic · tap at bridge point · tap at lower bout │ Outputs: Mode 1, Mode 2, Mode 5 frequencies (Hz) │ Bending stiffness EI (from bending rig) │ Species, thickness, density │ ▼ PHASE 2 ─ Installed Plate, No Strings (box closed, unstrung) │ │ Guitar on support · foam isolation · soundhole accessible │ Reference mic fixed · roving mic on 35-point grid │ Speaker-air excitation through soundhole │ Outputs: Transfer function H(f) at each grid point │ Coherence γ²(f) as quality gate │ ODS snapshot (operational deflection shapes) │ WSI curve (wolf suspicion index) │ ▼ PHASE 3 ─ Installed Plate, Strings at Pitch (fully strung) │ │ Same guitar · strings tuned to concert pitch │ Same reference mic · contact mic or air mic through soundhole │ Tap same grid points through soundhole │ Outputs: Mode 2 installed frequency under string prestress │ Wolf note identification (played notes vs body modes) │ Beat frequency measurement if wolf present │ ▼ THE BURT GAP — Phase 1 vs Phase 3 differential │ │ ΔMode2 = Mode2_Phase3 − Mode2_Phase1 │ Typically +15 to +45 Hz (assembly + string prestress) │ Varies by body style, brace pattern, string gauge │ ▼ CORRECTION ALGORITHM (target of the research program) │ │ Free-plate target = desired installed frequency − ΔMode2 │ Example: target 300 Hz installed → tune free plate to ~265 Hz │ ▼ Applied to next build → closes the loop
Phase 1

Free Plate

Before assembly. Plate suspended in air. Modes measured without boundary constraints or string load. The classic tap-tone measurement.

Phase 2

Installed · No Strings

Box closed, unstrung. Spatial ODS grid reveals mode shapes under assembly constraints. Quantifies the boundary condition shift from free to installed.

Phase 3

Installed · Strung

Final state at pitch. String prestress adds membrane stiffening to the top. This is the operating condition that produces wolf notes.

The Loop

What Closes It

Measuring the same instrument across all three phases on multiple builds establishes ΔMode2 as a predictable function of body style, brace pattern, and string gauge. Once that function is known, free-plate tuning targets can be corrected for the installed state — and wolf notes become a design choice rather than a post-assembly accident.

07Phase 1 — Free Plate Tap Tone

Protocol

ElementSpecification
Specimen stateGraduated plate, before assembly. Braces glued. F-holes or soundhole cut if applicable.
SupportFoam blocks at nodal intersection of modes 2 and 5. For guitar top: approximately over the bass-side soundhole, offset toward centerline.
ExcitationSharp tap with hard wooden dowel or pencil. One clean impulse — no double taps. Tap at bridge point (Mode 2 antinode) and at lower bout center.
Microphone20–30 cm from plate surface. Fixed position — mark it. Do not move between taps.
Capture2.5 seconds. 48 kHz sample rate. No AGC, no noise suppression, no enhancement.
Repeat3–5 taps per position. Discard outliers. Report mean and standard deviation.

What is being measured

Mode 2 (the cross-dipole, or X-mode in Caldersmith's nomenclature) is the primary measurement. It corresponds to the plate rocking side-to-side, with a nodal line running approximately through the waist. This is the mode most strongly coupled to the string's primary bending vibration and most likely to produce wolf notes when the installed frequency coincides with a commonly played note.

frequency ∝ √(stiffness / mass) // Chris Burt (American Lutherie #85, 2006) // Removing wood decreases both stiffness and mass, lowering frequency. // A note higher than target = plate too stiff = more wood to remove. // A note lower than target = graduated too far = irreversible. Phase 1 targets (free plate, 6-string acoustic, light gauge): Mode 1 (monopole, 0,0): ~80–120 Hz Mode 2 (cross-dipole, 1,0): ~150–200 Hz // primary target Mode 5 (ring mode, 0,2): ~280–350 Hz

Bending rig — MOE measurement

Run alongside Phase 1. Three-point bending on the same plate blank gives Young's modulus E_L (along grain) and E_C (cross grain). These values feed the plate design calculator in the Production Shop and establish the species-specific stiffness properties of that individual piece of wood — not a table average, but this specific plate.

08Phase 2 — Roving Grid ODS

Phase 2 maps the spatial behavior of the installed top plate across 35 points on a grid. It requires two simultaneous channels on a single interface with a shared clock. This is the phase that cannot be done with two separate USB microphones.

The two-channel requirement

Non-Negotiable — Shared Clock Transfer functions and coherence calculations require the phase relationship between channels to be meaningful. Two USB audio devices run on independent crystal oscillators that drift relative to each other. Even at a few parts per million of drift, the phase relationship becomes useless within seconds. Both channels must be captured by a single interface with one shared clock.

Measurement topology

Reference mic (fixed): placed near the soundhole, stationary throughout all 35 points Roving mic: moved point-to-point on the 35-point grid At each point: X_ref(f) = FFT of reference channel // normalization signal X_rov(f) = FFT of roving channel // response at this grid point Transfer function (H1 estimator): H(f) = G_xy(f) / G_xx(f) where G_xx = auto-spectrum of reference G_xy = cross-spectrum (roving × reference conjugate) Coherence (quality gate): γ²(f) = |G_xy(f)|² / (G_xx(f) × G_yy(f)) Range: 0 (no causal relationship) to 1 (perfect linear relationship) Threshold: γ² > 0.8 required to accept measurement at that frequency

09Phase 3 — Closed-Box Under String Prestress

Phase 3 is the measurement state that does not exist anywhere in the lutherie literature as a systematic practice. It is the installed plate under the actual operating load — strings at pitch, boundary conditions fully established, the instrument as a musician will play it.

Why this state is different

Free plate (Phase 1): - No boundary constraints (foam support ≈ free-free) - No in-plane compressive load - Modes are pure plate bending modes Installed, no strings (Phase 2): - Boundary constrained (kerfed lining, glue joint) - Top-to-back couple through sides - Mode frequencies shift UP ~15–30 Hz from Phase 1 Installed, strung (Phase 3): - Above constraints plus: - Bridge torque: ~2500 N·mm trying to rotate bridge forward - Saddle downforce: ~100 N vertical at bridge - In-plane compression forward of bridge (top is prestressed) - Mode frequencies shift UP additional ~5–15 Hz from Phase 2 Total Phase 1 → Phase 3 shift (Burt Gap): ΔMode2 ≈ 20–45 Hz (typical, varies by body style and string gauge)

How to measure Phase 3 through a soundhole

The challenge is that the soundhole is only 100mm diameter on a typical acoustic guitar, and the strings are in the way. The practical approach is to use a contact microphone mounted to the inside of the top at an antinode of Mode 2 (near the bass-side X-brace intersection), with a long wire exiting through the soundhole. The plate is tapped through the soundhole using a long dowel while the contact mic reads the response. A small USB audio interface accepts the contact mic on one channel and an air microphone on a second channel for coherence checking.

10Closing the Loop — The Wolf Note Prevention System

The loop connects the free-plate measurement (when you can still remove wood) to the installed behavior (when it is too late to remove wood without opening the box). Closing it requires empirical data from matched Phase 1 / Phase 3 measurements on the same instrument.

Current State — Loop Open

Before this research

Builder tunes free plate by ear to a note name. Closes the box. Strings to pitch. Discovers wolf note. Options: scrape through soundhole (imprecise, risky) or open the box (nightmare scenario).

Target State — Loop Closed

After calibration study

Builder measures free plate with tap-tone-pi. System applies Burt Gap correction for this body style. Builder tunes free plate to corrected target. Closes box. No wolf note.

The calibration study — the research program described in the preceding documentation — needs 4–5 matched builds per body style. The Chinese production instruments are the development platform. Their consistent overbuilding means corrections will be large and clearly measurable, making the calibration coefficients easier to establish on early builds.

11FFT Fundamentals — Theoretical Framework

What the FFT produces

Given N samples at sample rate f_s: Δf = f_s / N // frequency resolution (Hz per bin) f_max = f_s / 2 // Nyquist limit bins = N/2 + 1 // number of frequency bins (real signal) For guitar mode work at 48 kHz, 4096-sample block: Δf = 48000 / 4096 = 11.7 Hz // adequate for broad mode identification For 5 Hz mode resolution (Burt Gap precision): N ≥ 48000 / 5 = 9600 → use 16384 samples Δf = 48000 / 16384 = 2.9 Hz // adequate for Burt Gap measurement

Windowing — essential for impulse response work

The FFT assumes the signal is periodic within the analysis window. A tap tone impulse is not — it decays to silence. The Hanning window tapers the signal to zero at both edges, preventing spectral leakage. All tap tone work uses Hanning windowing. The decay portion of the impulse response is where the mode information lives — the window must be long enough to capture the full decay.

12Transfer Functions — Phase 2 Core

H(f) = Y(f) / X(f) // complex ratio: output to input at each frequency |H(f)| = magnitude (gain) at each frequency ∠H(f) = phase (delay) at each frequency Three estimators for different noise conditions: H1 = G_xy / G_xx // noise on output — best for speaker-driven excitation H2 = G_yy / G_yx // noise on input — best for impact hammer excitation Hv = √(H1 × H2) // noise on both channels equally For Phase 2 (speaker-air excitation, clean reference): Use H1. The excitation is controlled and the reference channel is clean. The roving channel picks up ambient noise → H1 is the correct estimator.

13Coherence — The Quality Gate

γ²(f) = |G_xy(f)|² / (G_xx(f) × G_yy(f)) Range: 0 (no linear relationship) → 1 (perfect linear relationship) Interpretation: γ² > 0.90 Excellent — measurement is trustworthy γ² 0.70–0.90 Good — usable with care γ² 0.50–0.70 Marginal — increase averaging or check setup γ² < 0.50 Poor — result must be discarded Causes of low coherence: - Ambient noise contaminating the roving channel - Nonlinear response (plate response is linear; strings are not) - Multiple signal paths (room reflections interfering) - Insufficient averaging - Extraneous vibration from HVAC or machinery Phase 2 quality gate: γ² < 0.80 at a frequency → that frequency bin is flagged invalid for that grid point. It is NOT averaged into the ODS.

14Plate Vibration Modes — The Targets

Guitar top plate modes follow Caldersmith's nomenclature, which Phil Banks also uses in the FEA article (American Lutherie #18, 1989). Mode shapes are labeled (m, n) where m is the number of nodal lines across the grain and n is the number along the grain.

ModeCaldersmith LabelShapeFree Plate TargetSignificance
Mode 1(0,0) monopoleWhole plate moves in phase — no nodal lines80–120 HzCoupled to Helmholtz air resonance; bass response
Mode 2(1,0) cross dipolePlate rocks side-to-side; one nodal line along grain150–200 HzPrimary graduation target; most wolf-prone mode
Mode 5(0,2) ring modeFour-quadrant pattern; two nodal lines cross280–350 HzCross-grain stiffness indicator; secondary check

The nodal intersection of Mode 2 and Mode 5 is the suspension point used in the classical tap-tone technique (Burt's thumb-and-finger method). Suspending at this point damps neither mode, allowing both to be excited with a single tap.

15Wolf Note Physics — The Coupled Oscillator

A wolf note occurs when string frequency ω_s ≈ body mode frequency ω_b. Coupled oscillator characteristic equation: λ±(ξ) = [1 + ξ² ± √((1 − ξ²)² + 4Ω²ξ²)] / 2 Where: ξ = ω_s / ω_b // detuning parameter (string/body frequency ratio) Ω² = κ² / (4ω_b²·m_b) // dimensionless coupling strength λ = ω² / ω_b² // normalized eigenfrequency squared At resonance (ξ = 1): Minimum gap: Δλ = 2Ω // the avoided crossing Beat frequency: f_beat ≈ Ω × f_body Audible symptoms by beat frequency: 1–4 Hz: growling, pulsating tone (most objectionable) 4–10 Hz: warbling, uneven sustain >10 Hz: roughness, less distinct beating Wolf masked (unresolvable) when: f_beat < (γ_lower + γ_upper) // peaks merge within combined linewidths

16The Burt Gap — Why the Loop Exists

Chris Burt (American Lutherie #85, 2006) explicitly lists what free-plate tap tones do not include: being glued to a rib structure, the compression of strings on the top, and the tension of the back under load. He is aware of the gap but does not model it. Phil Banks' FEA work (American Lutherie #18, 1989) does not include string forces either — he derives purely structural modal properties from the free plate.

The Burt Gap is the systematic offset between free-plate Mode 2 frequency and the same plate's Mode 2 frequency after assembly and stringing. It consists of two components:

ComponentMechanismTypical Magnitude
Assembly shiftBoundary condition change: free edge → constrained by kerfed lining and sides. The constraint stiffens the plate's effective bending resistance.+15–30 Hz
Prestress shiftString tension creates in-plane compressive load on the top plate forward of the bridge. This membrane stiffening raises out-of-plane bending modes slightly.+5–15 Hz
Total Burt GapPhase 1 → Phase 3 difference+20–45 Hz typical
The Research Question How consistent is ΔMode2 across builds of the same body style with the same brace pattern and string gauge? If the standard deviation is small relative to the mean — say, ±5 Hz on a mean shift of 30 Hz — then the correction is reliable enough to use as a design target. The Chinese production instruments are the controlled subjects for establishing this empirically. The tap-tone-pi pipeline is the measurement infrastructure. The study is the missing link.

18Measurement Boundary

This system measures and records. It does not interpret, score, grade, or advise. This distinction is architectural — it is enforced by CI guards in the codebase that prevent advisory logic from entering the measurement layer. The reasons are practical, not philosophical: measurement and interpretation require different validation standards, different uncertainty budgets, and different update cycles. Keeping them separate means each can be improved independently.

Interpretation of the measurements — what to do about a wolf note candidate, whether a Mode 2 frequency is appropriate for a given instrument design, what brace modification is indicated — is the function of the Production Shop calculators and the builder's judgment. The tap-tone-pi pipeline provides the evidence. The Production Shop provides the analysis framework. The builder closes the loop.
