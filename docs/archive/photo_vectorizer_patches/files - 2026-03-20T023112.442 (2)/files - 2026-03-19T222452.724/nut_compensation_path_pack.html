00Nut Systems Compared

TRADITIONAL NUT ZERO FRET ─────────────── ────────────── Open string: nut → saddle zero-fret → saddle ┌────────┐ ┌──────┬───────────┐ │ NUT │ ← sets height, │ NUT │ ZFRET │ │ (bone) │ intonation, │(guide│(wire │ └────┬───┘ spacing │ only)│ at pos 0)│ │ └──────┴─────┬─────┘ │ ← nut-to-fret-1 │ ← fret height ──────────────┼────────────────── ─────────────┼───────────── [fret 1] │ [fret 1] │ Open string height: set by nut slot depth set by zero fret crown height Open string material: bone / TUSQ / etc. nickel-silver (same as all frets) Nut role: height + spacing + inton. spacing only (height non-critical) Slot depth: critical — sets intonation non-critical — just a guide slot Wear: nut wears, easy to replace zero fret wears — fret job to fix Compensation: nut setback 0.005–0.020" very small (≈0.003–0.007")

The critical practical difference for the builder: on a traditional nut, the nut slot depth controls open string height and directly affects fret 1 intonation. This must be cut precisely. On a zero-fret instrument, the open string height is set by the zero fret crown — same as every other fret — and the nut is cut just deep enough to guide strings without contacting them.

01Source Honesty — Three Categories of Numbers

This module uses three categories of mathematics with very different epistemic status. They are distinguished here rather than treated as equivalent.

FormulaTypeUncertaintySource
12-TET fret positions Exact Zero — pure mathematics Euclidean geometry / equal temperament derivation
Saddle compensation Empirical ±15% Gore & Gilet Vol. 1 — derived from string mechanics, calibrated against measured instruments
Nut setback (traditional) Empirical ±30% Gore & Gilet methodology + luthier practice consensus. Setup-specific.
Zero fret residual offset Empirical ±50% Luthier practice. Limited published data. Treat as order-of-magnitude only.
Why the Uncertainty Is Large for Nut Compensation The nut compensation depends on action height at the nut, string stiffness, and playing style — all setup-specific variables. A guitar set up for fingerstyle with low action needs less nut compensation than the same guitar set up with high action for slide playing. The values in the tables are typical estimates for standard acoustic setups. The correct approach on precision work is to use these values as starting points and verify by checking fret 1 intonation against the 12th fret harmonic after installation.

02Fret Positions — 12-TET

These are exact. No empirical constants. No uncertainty. Results are deterministic for any scale length input. Exact Math

d_from_saddle(n) = L / 2^(n/12) // distance from saddle to fret n d_from_nut(n) = L - d_from_saddle(n) Key identity: fret 12 is exactly half scale length. d_from_saddle(12) = L / 2^(12/12) = L / 2^1 = L / 2 ← exact, always Example — Martin OM 25.4" scale: Fret 0 (nut): 0.000" (0.000 mm) Fret 1: 1.4256" (36.210 mm) Fret 2: 2.7712" (70.388 mm) Fret 3: 4.0412" (102.647 mm) Fret 5: 6.3715" (161.836 mm) Fret 7: 8.4475" (214.567 mm) Fret 12: 12.7000" (322.580 mm) = exactly half scale The formula L/2^(n/12) guarantees that each fret produces exactly one semitone above the previous fret in equal temperament. This is a mathematical property, not an approximation. The "rule of 18" (dividing remaining length by 17.817) is an approximation of this formula — do not use it in CNC work.

03Saddle Compensation

Empirical ±15% — Gore & Gilet Vol. 1 methodology, calibrated against measured instruments.

C_saddle ≈ k × gauge_diameter k = 0.80 for wound strings // heavier gauge, more stiffness k = 0.60 for plain strings Example — Martin OM 25.4", light gauge .012: Low E (.053" wound): 0.0424" (1.08 mm) saddle setback G (.024" wound): 0.0192" (0.49 mm) High e (.012" plain): 0.0072" (0.18 mm) Source: Gore & Gilet derive this from string tension + bending stiffness mechanics. The k coefficients are fits to measured instruments. Uncertainty ±15% — the correct saddle position should be verified by intonation check after installation.

04Traditional Nut Setback

Empirical ±30% — Gore & Gilet methodology + luthier practice consensus.

C_nut = C_saddle × k_nut k_nut = 0.22 heavy wound strings (Low E, A) k_nut = 0.28 light wound strings (D, G wound) k_nut = 0.35 plain strings (B, High e) Example — Martin OM 25.4", light gauge .012: Low E (.053"): C_saddle=0.0424" × 0.22 = 0.0093" (0.24 mm) setback G (.024"): C_saddle=0.0192" × 0.28 = 0.0054" (0.14 mm) setback High e (.012"): C_saddle=0.0072" × 0.35 = 0.0025" (0.06 mm) setback Uncertainty ±30% — this is an estimate, not a precision specification. Correct value is setup-specific: action height, string brand, playing style all shift it. Use these values as starting points. Verify with electronic tuner: fret 1 should match the pitch of the open string + one semitone after nut compensation is set.
Source Transparency The k_nut factors (0.22–0.35) are not published in a single authoritative reference. They are calibrated from the Gore & Gilet methodology and the luthier practice consensus that emerges from sources like Siminoff, StewMac instructional content, and GAL technical discussions. The dominant physical mechanism is bending stiffness at the short open-string segment — not the Pythagorean string-stretch model, which gives values approximately 20× too small. The ±30% uncertainty reflects genuine scatter in the underlying data.

05Zero Fret Position and Compensation

Empirical ±50% — limited published data. Zero fret residual offset is small enough that most builds place the zero fret at the theoretical position (no offset).

Zero fret position = theoretical nut position (fret 0, distance L from saddle) Standard builds: offset = 0.000" (no compensation) Precision builds: offset ≈ C_nut_traditional × 0.30 Rationale for near-zero offset: Open string height = zero fret crown height ≈ 0.045" (medium 6160) Fret 1 crown height = same fret wire = 0.045" Depression when fretting at 1 = action above fret 1 only → bending stiffness mismatch is much smaller than traditional nut → residual compensation is small (within tuning tolerance for most builds) Uncertainty ±50% on the 30% residual factor. In practice, place the zero fret at the theoretical position and verify intonation. Nut position behind zero fret: The nut sits approximately 3/8" (9.5mm) behind (headstock side of) the zero fret. Nut slot depth: string bottom should sit 0.005–0.010" ABOVE zero fret top. Strings must ride on zero fret — if they contact the nut, the zero fret is bypassed.

06Why Compensation Exists — The Physical Mechanism

Two distinct physical effects cause fretted notes to sound sharp relative to the theoretically expected pitch. Understanding which effect dominates which part of the scale is key to understanding why nut and saddle compensations differ.

Effect 1 — string stretching from fretting action

When a string is pressed to a fret, it is deflected from its resting position. This deflection stretches the string slightly, increasing tension. Increased tension = higher frequency = sharp pitch. The stretching effect is proportional to: (deflection_height)² / (2 × lever_arm_length) This effect is LARGEST at the SADDLE end because: - The lever arm (vibrating string length) is longest there - The deflection is proportional to the action height, which is largest near the body This is the dominant mechanism for saddle compensation.

Effect 2 — bending stiffness of the string

A real string has finite stiffness (it resists bending, like a thin rod). A stiffer string vibrates at a slightly higher frequency than a pure tensioned string of the same length. The effect is larger for shorter vibrating lengths. The bending stiffness correction scales approximately as: Δf/f ∝ (EI) / (T × L²) where: EI = bending stiffness (proportional to gauge^4 approximately) T = string tension L = vibrating length For the OPEN string: L = full scale length (smallest correction) For FRET 1: L = scale_length / 2^(1/12) ≈ scale_length × 0.9439 The bending stiffness correction for fretted notes is slightly larger than for the open string — the string is "stiffer per unit length" when the vibrating segment is shorter. This makes the fret 1 note sharp relative to the open string. This is the dominant mechanism for NUT compensation. KEY RESULT: simple Pythagorean stretching predicts nut compensation of ~0.001–0.003". Bending stiffness predicts ~0.005–0.015". The empirical values seen in practice match bending stiffness, not stretching.

07Open String Tone Character

The Open String Argument for Zero Fret The principal reason professional players sometimes prefer zero-fret guitars is consistency: an open G chord has the same attack character as a closed G chord on fret 3. On a traditional nut, the bone or TUSQ contact produces a slightly different initial transient than the fret wire contact of a stopped note. This difference is subtle and debated, but it is real and detectable in close-mic recording. Classical and fingerstyle players tend to notice it more than strummers.

08Nut Slot Depth — Traditional Nut

Slot depth = nut_blank_height - fret_crown_height - target_clearance Target clearance (string bottom above fret 1 crown): Plain strings: 0.012" (0.30 mm) Wound strings: 0.017" (0.43 mm) Example — 0.200" nut blank, medium fret (crown 0.045"): High e (.012" plain): 0.200 - 0.045 - 0.012 = 0.143" depth from top Low E (.053" wound): 0.200 - 0.045 - 0.017 = 0.138" depth from top Source: Siminoff; StewMac; shop practice. These are minimum slot depths — too shallow = string too high = sharp fret 1. Too deep = string buzzes on fret 1 or lower frets. Cut incrementally. Verify with electronic tuner. Each pass is irreversible.
Zero Fret Nut — Different Requirement On a zero-fret guitar, cut the nut slots so the string bottom sits 0.005–0.010" ABOVE the zero fret crown. The string must contact the zero fret, not the nut. If the nut is cut too deep, strings bypass the zero fret entirely and the instrument becomes a traditional nut guitar with wrong nut position.
