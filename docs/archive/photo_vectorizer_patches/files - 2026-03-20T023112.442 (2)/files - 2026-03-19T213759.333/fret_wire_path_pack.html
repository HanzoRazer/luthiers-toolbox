00Fret Wire Anatomy

Every fret wire profile has four dimensions that govern its behavior in use. Width and height define the crown — what the player feels and what wears. Tang depth and width define the portion buried in the fretboard slot.

Crown Width (W) ◄──────────────────────► ╭─────────────╮ ← Crown top ╱ ╲ ╱ ╲ ▲ ╱ ╲ │ Crown Height (H) ─────────────╱─────────────────────╲──┤──── fretboard surface │ │ ▼ │◄───────────────────►│ ← W (same as crown) │ │ │ ╔═════════════╗ │ │ ║ TANG ║ │ ▲ │ ║ ║ │ │ Tang Depth (D) │ ║◄─── T ─────►║ │ ▼ └──╚═════════════╝────┘ ← slot bottom Crown sits above the fretboard. Tang is buried in the slot. The barbs on the tang sides grip the wood and prevent the fret from lifting.

The crown presents a curved surface to the string. That curvature — described by the crown radius — determines how wide the string contact zone is, which directly affects intonation precision and wear rate per unit area.

02Core Formulas

Crown Radius — intonation contact width

R_crown = (W/2)² / (2H) + H/2 // chord-sagitta formula contact_width = 2 × √(2 × R × d − d²) where d = string contact depth ≈ 0.001" (half light-gauge string radius) Example — Tall-Narrow 6105 (W=0.090", H=0.055"): R_crown = (0.045)² / (0.110) + 0.0275 = 0.046" contact_width ≈ 19 thou // 0.019" — narrow, precise Example — Jumbo (W=0.110", H=0.055"): R_crown = (0.055)² / (0.110) + 0.0275 = 0.055" contact_width ≈ 21 thou // wider contact, slightly less precise Narrower contact zone means: + More precise intonation at any string position + String notches deeper into crown per stroke (faster groove formation) → Tall-narrow profile trades some longevity for precision

Leveling headroom

headroom = H - H_min H_min = 0.015" // minimum safe crown height (empirical) passes ≈ headroom / 0.006" // average material per pass Vintage Narrow (H=0.036"): headroom = 0.021" → ~3 passes Tall-Narrow 6105 (H=0.055"): headroom = 0.040" → ~6 passes The crown HEIGHT — not the width — determines leveling headroom. Medium-Jumbo (0.106" wide, 0.036" tall) has the same headroom as Vintage Narrow despite being much wider. Width is irrelevant to leveling longevity.

Action interaction

String clearance above crown = action_at_fret - H_crown A taller crown reduces required nut/saddle action: For same playing clearance of 0.015" above the crown: Vintage Narrow (H=0.036"): action needed = 0.036 + 0.015 = 0.051" Tall-Narrow (H=0.055"): action needed = 0.055 + 0.015 = 0.070" → Tall wire can be set up with lower nut/saddle while maintaining the same clearance above the fret. This is why tall wire "plays lower." The saddle and nut set the string height above the fretboard surface. The fret crown height adds to the effective playing height. Total string height above crown = (saddle/nut setting) - H_crown.

03Vintage Family

Wide, low profiles descended from mid-century Gibson and Fender specifications. The defining characteristic is a large crown-width-to-height ratio — the string sits in a shallow groove and the player's fingertip contacts the fretboard through the string. This is the "vintage feel" that players describe.

05Tall-Narrow and Jumbo

Tall profiles give the most material for leveling and allow the lowest action settings. The narrow crown on the 6105 is a different design philosophy from the jumbo — precision over surface area. The jumbo prioritizes maximum material and a broad feel.

Stainless Warning Do not attempt stainless fret installation without carbide or diamond tooling and prior experience. Standard fret files are destroyed on first contact. The work-hardening effect during hammering changes the material properties of already-installed frets — once seated, the fret is harder than it was in the bag. Heat generated during leveling can damage lacquer and nitro finishes. EVO Gold provides 2.7× the wear life of nickel-silver with none of these risks and is the better choice for most builders upgrading from standard wire.

08Action Interaction

The fret crown height is additive to string height. When a player "plays to the fret" — just enough pressure to touch the crown — the crown height sets the floor for effective action. Taller wire allows lower nut and saddle settings while maintaining the same playing clearance above the fret.

ProfileCrown HClearance @ 0.070" actionAction for 0.015" clearance
Vintage Narrow 62300.036" 0.034" 0.051"
Medium 61600.045" 0.025" 0.060"
Tall-Narrow 61050.055" 0.015" 0.070"
Practical Implication On a guitar set up to 0.070" action at the 12th fret (a common acoustic specification), tall-narrow wire leaves only 0.015" clearance above the fret crown. This is very low — aggressive players may produce fret buzz. But it also means the setup can legitimately lower the saddle to reduce string height because the tall fret is providing height the saddle previously had to supply. This is why identical action measurements feel different across instruments with different fret profiles.

09Fretboard Thickness Interaction

The tang must fit within the fretboard with at least 0.040" (1.0mm) of wood below the tang tip. For standard tang depths this is almost never the limiting factor on well-dimensioned fretboards, but it becomes relevant on very thin bound fretboards and thin modern electrics.

wood_below_tang = fretboard_thickness - tang_depth Minimum safe: 0.040" (1.0mm) Standard tang 0.050": Acoustic fretboard 0.240": 0.240 - 0.050 = 0.190" remaining ✓ OK Electric fretboard 0.200": 0.200 - 0.050 = 0.150" remaining ✓ OK Thin electric 0.180": 0.180 - 0.050 = 0.130" remaining ✓ OK Deep tang 0.055" (6105, Jumbo): Acoustic fretboard 0.240": 0.240 - 0.055 = 0.185" remaining ✓ OK Electric fretboard 0.200": 0.200 - 0.055 = 0.145" remaining ✓ OK Thin electric 0.180": 0.180 - 0.055 = 0.125" remaining ✓ OK Very thin 0.120": 0.120 - 0.055 = 0.065" remaining ⚠ marginal Standard instruments are always adequate. The check matters for non-standard builds — scalloped fretboards, very thin radiused boards, or boards thinned during refrets.
