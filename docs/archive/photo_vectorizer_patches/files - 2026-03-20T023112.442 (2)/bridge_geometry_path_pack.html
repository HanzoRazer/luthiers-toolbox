00Scope & Module Map

The break angle calculator (bridge_break_angle.py) already exists and handles string angle over the saddle crown. This path pack covers everything that calculator does not know: the physical bridge body, its geometry, and how its dimensions propagate to the break angle inputs.

Existing — break_angle.py
Saddle projection height · Pin-to-saddle distance · Slot offset · Break angle θ · Carruth 6° minimum
New — acoustic_bridge_calc.py
String spread nut↔saddle · Saddle crown radius · Slant angle · Slot location · Pin hole placement · Belly profile · Bridge plate

The two modules connect through break_angle_inputs: the belly profile computes slot_offset_mm, which is the previously hard-coded default in BreakAngleInput.

§1String Spacing: Nut vs Saddle

The nut and saddle use different string spacings. The nut is narrow (comfort for the fretting hand); the saddle is wider (string spread required by the body geometry). Strings diverge linearly from nut to saddle.

1.1 Per-String Spacing

p_nut = N_ee / (n − 1) // mm per string at nut p_saddle = S_ee / (n − 1) // mm per string at saddle
Assumes uniform spacing (equal center-to-center). Classical guitars use non-uniform spacing but the same formula applies per-interval.

1.2 String Position Offsets from Centerline

Treble strings are assigned negative Y values; bass strings positive. For string i (0 = high E, n−1 = low E):

y_nut[i] = −N_ee/2 + i × p_nut y_saddle[i] = −S_ee/2 + i × p_saddle

1.3 String Taper Half-Angle

Each outer string diverges from the centerline at a small angle. The half-angle is:

φ = arctan( (S_ee − N_ee) / (2 × SL) )
Martin OM (SL=648mm, N_ee=34.5mm, S_ee=55.0mm): φ = arctan(20.5 / 1296) = 0.906°

§2Saddle Crown Radius

The saddle top is shaped as a cylindrical arc matching the fretboard radius. This keeps string action consistent across all strings. On a radiused fretboard, the center strings sit lower than the edge strings — the saddle must replicate this profile.

2.1 Crown Height Formula

The height of a circular arc above the chord at lateral position y (measured from centerline) is:

h(y) = R − √(R² − y²) // height above arc chord at y Height of edge string above saddle geometric center: h_edge = R − √(R² − (S_ee/2)²) Drop from edge to any string i: Δh[i] = h(y_saddle[i]) − h_edge // negative = lower than edge Crown height (total span): crown = |Δh[center]| = h_edge − h(0) = R − √(R² − (S_ee/2)²)
At center string (y=0): h(0)=0, so crown = h_edge. The center strings sit h_edge mm below the edge strings on the saddle arc.

§3Saddle Slant Angle

The saddle slot is cut at an angle so that wound strings (which require more intonation compensation — their saddle contact must be farther from the nut) achieve the correct compensation without an individually shaped saddle for each string. The slot angle equals the arctangent of the compensation spread over the string spread.

3.1 Derivation

Let Δc_max = compensation of most-wound string (e.g., E6 = 3.0 mm) Let Δc_min = compensation of least-wound string (e.g., E1 = 1.0 mm) Compensation spread: ΔC = Δc_max − Δc_min = 2.0 mm // for standard 6-string Slant angle (positive = bass end farther from nut): α = arctan( ΔC / S_ee ) = arctan( 2.0 / 55.0 ) = 2.08° End-to-end slot offset (how much the slot deviates laterally): δ_slot = S_ee × tan(α) = ΔC = 2.0 mm // exact identity
The end-to-end slot offset equals the compensation spread exactly. The angle is simply the arctangent that produces this. The slot length is S_ee + 2×overhang ≈ 59 mm for a standard bridge.
CNC Note
The saddle slot must be routed at slant angle α. For the BCAM 2030A, this is a single-pass operation with the workholder rotated α degrees from the bridge centerline. A zero-angle slot produces a systematic intonation error: treble strings will be flat, bass strings sharp, with no per-saddle correction possible on a bone blank.

§4Saddle Slot Location

The saddle slot must be positioned on the bridge body so that adequate wood remains on both sides. The rear margin (between slot and back bridge edge) is the structural shear zone that resists the saddle being pushed backward by string tension.

4.1 Slot Depth

depth = blank_height − h_proj = 12.0 − 2.5 = 9.5 mm // standard bone blank, 2.5 mm projection Seat depth (saddle seated in slot): seat = depth = 9.5 mm // must be ≥ 3.0 mm minimum

4.2 Location Constraints

Let x_ctr = saddle centerline from back edge of bridge Rear margin (back edge to slot rear wall): x_rear = x_ctr − slot_width/2 // must be ≥ 6 mm Forward margin (slot front wall to pin row): x_fwd = d_ps − slot_width/2 // must clear pin holes Martin OM standard: x_ctr = 8.0 mm → x_rear = 6.5 mm ✓
6 mm rear minimum established from shear area analysis: string tension component pushing saddle backward is absorbed by the wood cross-section between slot and back edge. Below 6 mm, the cross-section is insufficient for sustained string loads.

4.3 Slot Length

L_slot = S_ee + 2 × overhang = 55.0 + 2 × 2.0 = 59.0 mm The saddle overhangs the outer strings by 2 mm per side to allow for string-path geometry and saddle end finishing without chipping.

§5Bridge Pin Holes

5.1 Y-Positions (Lateral)

Pin holes are at exactly the same lateral positions as the corresponding saddle strings:

y_pin[i] = y_saddle[i] // each pin is directly behind its string For Martin OM (S_ee = 55.0 mm, 6 strings): y_pin = [−27.5, −16.5, −5.5, +5.5, +16.5, +27.5] mm

5.2 X-Position (Neck-to-Tail)

x_pin = saddle_x − d_ps // pin row is d_ps forward of saddle Standard values: Martin OM/D: d_ps = 10.5 mm Taylor: d_ps = 10.0 mm Gibson J-45: d_ps = 9.5 mm

5.3 Hole Geometry

Diameter at surface: 5.5 mm // standard bridge pin taper Taper half-angle: 3.0° Total depth: 22 mm // through bridge + plate + into top = bridge height (10) + plate thickness (4) + top penetration (8)
CNC Op Note
Pin holes are drilled with a tapered reamer at 3° half-angle, not a straight drill. A straight drill produces a cylindrical hole that accepts pins only at the surface — the pin sits on its shoulder, not along its taper. Tapered holes allow proper pin seating and the slight side-loading that secures the pin under string tension.

§6Bridge Belly Profile

The belly is the raised ramp between the pin holes and the saddle slot. It serves two functions: (1) it prevents the string from cutting into the bridge wood at the pin hole exit; (2) it imparts a slight upward angle to the string as it approaches the saddle, contributing to the effective break angle.

6.1 Profile Function

The belly follows a cosine bell curve, smooth at both entry (pin hole) and exit (saddle face):

z(x) = h_belly × [1 − cos(π × x / d_ps)] / 2 Where: x = distance from pin center toward saddle [0, d_ps] h_belly = peak belly height above bridge surface (mm) d_ps = pin-to-saddle distance (mm) Properties: z(0) = 0 // at pin center — flush with bridge surface z(d_ps/2) = h_belly // peak at midpoint z(d_ps) = h_belly // at saddle face — maximum height // (the cosine function reaches its maximum at x=d_ps)
The cosine curve was chosen (vs. linear ramp) because it is tangent to the bridge surface at x=0, preventing a sharp corner that would cause string fatigue at the pin hole exit. The curve is smooth enough that a cabinet scraper can profile it from a flat-topped bridge blank.

6.2 Effective Slot Offset

At the saddle face (x = d_ps), the cosine function reaches its maximum. The string exits the belly at height h_belly above the bridge surface. However, the practical effective offset is less than the full belly height because the string's contact with the saddle begins slightly inboard of the saddle face. Empirically:

δ = 0.30 × h_belly // effective string elevation at saddle crown contact For h_belly = 2.0 mm: δ = 0.60 mm This value is slot_offset_mm in BreakAngleInput.
The 0.30 coefficient is conservative. The string wraps slightly over the belly peak and descends to meet the saddle crown at a point above the bridge surface. Physical measurement on reference instruments gives δ ≈ 0.4–0.8 mm for a 2 mm belly; 0.30 × h provides a consistent lower bound.

§7Connection to Break Angle Calculator

The acoustic bridge calculator produces a dictionary of inputs that feed directly into bridge_break_angle.py. All five inputs are now derived from physical geometry rather than entered as empirical defaults.

break_angle_inputs = { pin_to_saddle_center_mm: d_ps = 10.5, slot_offset_mm: δ = 0.60, // from belly §6.2 saddle_projection_mm: h_proj = 2.5, saddle_slot_depth_mm: depth = 9.5, saddle_blank_height_mm: blank_h = 12.0, }

7.1 Break Angle Calculation

Effective horizontal distance: d_eff = d_ps − δ = 10.5 − 0.60 = 9.90 mm Break angle: θ = arctan( h_proj / d_eff ) = arctan( 2.5 / 9.9 ) = 14.17° Rating: adequate (Carruth minimum: 6°; mechanical limit: 38°)
What Changed from the Default
The original BreakAngleInput default for slot_offset_mm was 1.2 mm (empirical estimate for a slotted pin hole). The belly-derived value is 0.6 mm — a difference of 0.6 mm, which changes the effective distance from 9.3 mm to 9.9 mm and the resulting angle from 15.03° to 14.17°. Both are adequate; the belly-derived value is physically traceable.

§8Bridge Plate Sizing

The bridge plate is glued to the inside of the top, under the bridge. It distributes the string load (transmitted via bridge pins pulling down on the top) over a larger area of the top plate, reducing stress concentration at the pin holes.

8.1 Dimensions

L_plate = L_bridge + 2 × margin = 155 + 20 = 175 mm W_plate = S_ee + 2 × margin + 20 = 55 + 20 + 20 = 95 mm t_plate = max(3.0, 1.2 × t_top) = 3.0 mm // for t_top = 2.5 mm
The +20 mm extra width beyond string spread + margin ensures the plate spans the full X-brace intersection zone. The X-brace crosses at approximately ±35 mm from centerline; the plate must anchor both brace legs to prevent the bridge plate from pivoting under string load. Without X-brace coverage the plate acts as a lever rather than a pad.

8.2 Grain Direction

Bridge plate grain must run parallel to the top grain (neck-to-tail axis). This aligns the stiffer axis of the plate with the primary load direction (string tension pulls pins toward the bridge with a downward component). A cross-grain plate is common in production instruments but provides approximately 15× less stiffness in the critical direction (E_L vs E_C for sitka spruce).
