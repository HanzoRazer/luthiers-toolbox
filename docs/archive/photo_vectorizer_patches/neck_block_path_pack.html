00Scope & Physical Model

The neck block and tail block are the internal structural anchors at the neck and tail ends of the guitar body. Their dimensions determine glue surface area, which determines joint strength under string load. This calculator sizes both blocks from first principles rather than convention alone.

Neck Block — Structurally Critical Carries the full string tension as shear, plus a bending moment from the string height above the joint. Joint type (dovetail, bolt-on, tenon) determines the effective shear area — which is never the full block face.
Tail Block — Geometry Driven Carries essentially no string tension. String force is directed toward the nut — the tail block is in the opposite direction. Dimensions are set by body geometry with a minimum depth for the end pin hole.

The fundamental result: actual block face areas (5,000–9,000 mm²) are 25–40× the structural minimum. The neck block does not fail by face-shear under any normal string load. What matters is the joint type, which reduces the effective shear area by 28–100% depending on geometry.

§1String Tension

String tension is the input to all downstream structural calculations. Values are from D'Addario published tension data at concert pitch (A=440 Hz, standard tuning).

1.1 Per-String and Total Tension

T_total = Σ T_i // sum over all n strings Light gauge (.012–.053): T = [71.7, 49.0, 63.1, 72.6, 78.7, 84.4] N // E1→E6 T_total = 419.5 N = 42.8 kgf Medium gauge (.013–.056): T_total = 468.7 N = 47.8 kgf Nylon (normal tension): T_total = 389.0 N = 39.7 kgf Source: D'Addario String Tension Charts (published per gauge set at standard pitch). These are static tensions at rest pitch — dynamic loading during playing is higher but brief; static tension governs glue creep and long-term joint integrity.

§2Shear Force at Neck Joint

String tension acts approximately along the neck axis. At the neck joint face this becomes a shear force. The neck angle is small enough that cos(α) ≈ 1.

V = T_total × cos(α) = 419.5 × cos(1.5°) = 419.4 N // cos(1.5°) = 0.9997 — neck angle negligible The neck angle contributes a small normal force component (T·sin(α) ≈ 11N) perpendicular to the joint face. This is a compression load on the joint, not a shear, and is always beneficial — it clamps the joint closed.

§3Bending Moment and Couple Force

The strings are elevated above the neck joint pivot point by approximately 65 mm (neck heel thickness + fretboard). This eccentricity creates a bending moment that tends to rotate the neck forward. The moment is resisted by a compression/tension couple acting through the neck block.

3.1 Moment Derivation

M = T_total × h_neck = 419.5 N × 65 mm = 27,268 N·mm = 27.3 N·m Couple force (compression at fingerboard, tension at back): F_couple = M / d_side For Martin OM (d_side = 95mm): F_couple = 27268 / 95 = 287 N = 29 kgf This couple force acts as a vertical load on the top plate (compression) and back plate (tension) through the block's glue surfaces on each side. The glue area on the side surface (height × depth) carries this load.

3.2 Side Depth Effect on Couple Force

Deeper sides reduce the couple force. This is why deeper-bodied instruments have a structural advantage at the neck joint — the longer moment arm spreads the bending load over a larger Z dimension.

§4Minimum Glue Area from First Principles

4.1 Shear Strength of Wood

The limiting factor is wood shear strength, not glue strength. Both hide glue and Titebond joints exceed wood shear strength when properly prepared — the wood fails first. Values from USDA Wood Handbook Table 4-3 at 12% MC:

Speciesτ (MPa)τ·η (MPa)Common Use
Hard Maple10.08.50Neck blocks, bindings
Mahogany9.58.08Standard neck block
Walnut9.58.08Neck blocks
Cherry8.97.57Occasional
Sitka Spruce5.54.68Lightweight builds
Basswood5.54.68Production instruments

4.2 Minimum Area Formula

Required glue area for shear: A_min = V × SF / (τ × η) For mahogany, light gauge, SF = 4.0: A_min = 419.4 × 4.0 / (9.5 × 0.85) = 1677.6 / 8.075 = 208 mm² For basswood (worst case wood, heavy strings, SF=4): A_min = 468.7 × 4.0 / (5.5 × 0.85) = 401 mm² Safety factor 4.0 from NDS (National Design Specification for Wood Construction). Glue efficiency η = 0.85 accounts for stress concentrations at joint edges and imperfect wood-to-wood contact in real joints (as opposed to ideal test specimens). The actual joint areas (5000–9000 mm²) are 25–40× these minimums.
Key Result The structural minimum glue area (208–401 mm²) is vastly exceeded by actual block face areas (5,000–9,000 mm²). Face-shear is not the limiting failure mode for any standard block geometry. The critical question is joint type and fitting quality, not block size.

§5Joint Type — Effective Shear Area

The block face area is not the shear area for most neck joint types. Only the surfaces that are actually in contact and bonded carry shear. The effective area fraction converts block face area to the actual shear-carrying area.

A_effective = A_face × f_joint Joint safety multiple: N_joint = A_effective / A_min
Joint Typef_jointShear MechanismOM N_joint
Dovetail0.28 Two angled cheeks: 2×(50×20) = 2,000 mm² on a 7,080 mm² face 9×
Bolt-On0.35 Heel face glue + bolt-preload friction (~150N/bolt) 11×
Mortise-Tenon (NT)0.65 All four tenon faces; largest removable joint shear area 22×
Spanish Heel1.00 Integral — sides slot into block, no discrete shear plane 26×
Butt Joint (glued)1.00 Full face area is the shear plane 26×
Dovetail Note The dovetail f_joint = 0.28 means only 28% of the block face is an active shear surface. Even so, the dovetail joint provides 9× the structural minimum — failure by shear requires neglect or systematic joint misfitting. A poorly fitting dovetail (gap-fit, insufficient glue penetration) can reduce the effective area fraction below 0.10.

§6Neck Block Dimensions

Block dimensions are set by geometry (conventional proportions of the body) then verified against structural requirements. The geometry always governs — structural requirements are far below conventional dimensions.

Block height (Z axis) = side depth at neck end H = d_side // flush with top and back edges Block width (Y axis, treble-to-bass): W = upper_bout_width × 0.29 = 290 × 0.29 = 84 mm // Martin OM reference Block depth (X axis, neck-to-tail): D = max(joint_depth + 5mm margin, 65mm) = max(75 + 5, 65) = 80 mm // dovetail joint type Face area: A_face = W × D = 84 × 80 = 6,720 mm² Width fraction 0.29 derived from measured reference instruments: Martin OM: 84/290 = 0.290 Martin D-28: 90/292 = 0.308 Martin 000: 80/279 = 0.287 Mean = 0.295 → use 0.29

§7Tail Block — Geometry Driven

The tail block carries no meaningful string tension. String force is directed toward the nut — the tail is in the opposite direction. The tail block is loaded only by:

  • End pin: strap + guitar weight ≈ 50 N downward
  • Side glue: humidity differential shear (negligible for sizing)
  • Back brace ends: small bending from back plate curvature
Tail block height = side depth at tail end H = d_tail // flush with top and back Tail block width: W = lower_bout_width × 0.16 = 381 × 0.16 = 61 mm // Martin OM reference Tail block depth (minimum — end pin geometry): D_min = pin_radius + min_wood_around_pin + margin = 4.75 + 8.0 + 4.0 = 16.75 mm D = max(D_min, 22 mm) = 22 mm // conventional minimum Width fraction 0.16 from measured instruments: Martin OM: 64/381 = 0.168, D-28: 70/397 = 0.176, 000: 60/368 = 0.163

§8End Pin Bearing Check

End pin load: guitar weight + strap tension ≈ 50 N (conservative) Bearing area (half-pin circumference × block depth): A_bearing = π × D_pin / 2 × D_block × 0.5 = π × 9.5 / 2 × 22 × 0.5 = 164 mm² Bearing stress: σ_bearing = F_strap / A_bearing = 50 / 164 = 0.305 MPa Allowable bearing (wood across grain): ~3.0 MPa Safety multiple: 3.0 / 0.305 = 9.8× ✓ adequate End pin bearing is trivially within limits for any standard wood. The 22mm minimum depth is set by drilling and fitting practice, not structural limits.
