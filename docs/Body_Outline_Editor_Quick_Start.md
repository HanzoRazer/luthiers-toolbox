# Body Outline Editor — Quick Start Guide
## 5 Minutes to Your First Outline

**Access:** [INTERNAL_ACCESS_NOTE]

---

## Step 1: Launch the Editor

[INTERNAL_ACCESS_NOTE]

When the editor opens, you'll see a default dreadnought-like outline.

---

## Step 2: Choose Your Mode

A dialog appears on launch:

- **Default Mode** — Start with a dreadnought-like outline (edit existing)
- **Empty Mode** — Start with nothing (build from scratch)

For your first outline, choose **Default Mode**.

---

## Step 3: Set Up Your Workspace

In the toolbar:

1. Set **Snap** to `1 mm` (dropdown)
2. Click **Mirror: OFF** to turn mirror mode ON (green button, dashed centerline appears)
3. (Optional) Click **Import Image** to add a reference photo

---

## Step 4: Move Nodes (Default Mode)

1. Click a white dot (node) to select it
2. Drag the node to a new position
3. Watch the grid snap — the node jumps to 1mm increments
4. The mirror node on the opposite side moves automatically

**Try nudging:** Select a node, press arrow keys to move by 1mm. Hold Shift for 0.1mm fine movement.

---

## Step 5: Shape the Curve

1. With a node selected, look for two orange diamond handles
2. Drag a handle to adjust the curve
3. **Alt+click** the node to toggle between smooth and sharp (cusp)

**Try this:** Select a node, drag a handle outward. The curve bulges. Drag the other handle — they move together by default. Hold Alt to move only one side.

---

## Step 6: Add More Detail

Click on the dashed curve line between two nodes — a new node appears. Drag it to refine the shape.

---

## Step 7: Export Your Outline

Click **Export DXF** in the toolbar.

Options you'll see:
- **Format:** R12 (most compatible) or R2004+ (modern)
- **Tessellation:** 20 points per curve (default is fine)
- **Layer naming:** Standard or CAM ready

Click Export. A DXF file downloads.

---

## Keyboard Shortcuts (Most Important)

| Key | Action |
|-----|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Delete` | Delete selected node |
| `Arrow keys` | Nudge node |
| `Shift+Arrow` | Fine nudge (0.1mm) |
| `Space+drag` | Pan canvas |
| `G` | Toggle grid |
| `M` | Measure distance |
