# Strategy Package Examples

This directory contains example strategy packages generated by the CAM-A5 assembly CLI.

## Contents

### `fret_slot_strategy_example/`

A complete strategy package for a 22-fret, 25.5" scale fretboard:

- `strategy.json` — Validated strategy with fret positions and operation parameters
- `review_packet.md` — Human-readable review document with sign-off checklist
- `manifest.json` — Package manifest with authority declarations

## Non-Execution Notice

These packages are **advisory documentation only**.

They do not:
- Generate G-code
- Authorize machine execution
- Replace operator judgment
- Bypass human review requirements

## Generating Your Own Package

```bash
python scripts/assemble_strategy_package.py your_strategy.json --out ./your_package
```

See [docs/strategy_packages/STRATEGY_PACKAGE_ASSEMBLY.md](../../docs/strategy_packages/STRATEGY_PACKAGE_ASSEMBLY.md) for full documentation.
