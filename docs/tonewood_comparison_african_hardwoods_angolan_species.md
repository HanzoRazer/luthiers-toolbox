<a id="milicia_excelsa"></a>
### Mvule / Iroko

**ID:** `iroko` · **Category:** hardwood · **Scientific name:** *Milicia excelsa* (Welw.) C.C.Berg · **Also known as:** Iroko, African Teak, Kambala, Odum, Mvule (Kenya/Tanzania)

| Property | Value |
|----------|-------|
| Specific Gravity | 0.64 |
| Density | 660 kg/m³ |
| Janka Hardness | 1260 lbf / 5610 N |
| Shrinkage (radial) | 2.8–3.5% |
| Shrinkage (tangential) | 3.8–5.4% |
| T/R Ratio | 1.4–1.5 |
| Grain | interlocked |
| Workability | good — some calcium carbite deposits can dull tools |
| Resinous | No |
| CNC Risks | Burn: low, Tearout: medium, Dust: medium (sensitizing) |
| Guitar Uses | body, neck, back/sides |

Contains calcium carbonate deposits (stone) that can chip carbide.
