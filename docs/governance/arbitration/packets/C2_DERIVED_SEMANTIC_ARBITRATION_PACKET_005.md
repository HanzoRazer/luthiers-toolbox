```
How should the repository govern systems that:
  - Produce semantic outputs
  - Constrain semantic paths
  - Influence semantic interpretation

  - Creating governance
```

C2-A through C2-D decomposed semantic surfaces (geometry, topology, provenance, continuity).

- Translators silently become definers

- Operational behavior hardens into ontology

```
Systems can exert semantic influence
without possessing semantic authority.

Semantic influence does not imply semantic legitimacy.
```

## 5. RISK-SEM-* Taxonomy

### 5.1 Risk Summary (7)

| Risk ID | Name | Severity |
|---------|------|----------|
| RISK-SEM-001 | Hidden Authority Emergence | CRITICAL |
| RISK-SEM-002 | Operational Canonization | CRITICAL |
| RISK-SEM-003 | Derivation Collapse | CRITICAL |
| RISK-SEM-004 | Propagation Amplification | HIGH |
| RISK-SEM-005 | Semantic Hardening | HIGH |
| RISK-SEM-006 | Transformation Mutation | HIGH |
| RISK-SEM-007 | Cache Authority Drift | HIGH |

### 5.2 Risk Descriptions

**RISK-SEM-001: Hidden Authority Emergence**
```
Derived system silently becomes de facto authority.
Mechanism: Outputs become widely consumed → treated as authoritative → original authority bypassed.
```

**RISK-SEM-002: Operational Canonization**
```
Operational constraints become canonical semantic law.
Mechanism: Constraint enforced as gate → "must pass" becomes "semantically correct" → operational = governance.
```

**RISK-SEM-003: Derivation Collapse**
```
Derived values collapse into authoritative values.
Mechanism: Value cached → provenance lost → consumer treats as authoritative.
```

**RISK-SEM-004: Propagation Amplification**
```
Semantic influence amplifies through propagation.
Mechanism: Value propagates → many consumers → "used by many" implies authority.
```

**RISK-SEM-005: Semantic Hardening**
```
Values harden into immutable truth through persistence.
Mechanism: Value cached → retrieved repeatedly → changes become "breaking" → hardened truth.
```

**RISK-SEM-006: Transformation Mutation**
```
Meaning mutates through transformation.
Mechanism: Transformed → subtle changes → transformed meaning treated as original.
```

**RISK-SEM-007: Cache Authority Drift**
```
Cached values drift toward authority status.
Mechanism: Cache becomes primary access → source accessed rarely → cache = de facto authority.
```
