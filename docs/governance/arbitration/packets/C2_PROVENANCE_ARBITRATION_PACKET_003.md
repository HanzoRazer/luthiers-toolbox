## 6. Collapse Risk Summary

### 6.1 Critical Risks

| Risk | Mechanism | Prevention |
|------|-----------|------------|
| All provenance → generic source | Schema simplification | Category structure |
| Derivation → authority | Silent escalation | Provenance persistence |
| Sandbox → canonical | Escape without gate | Gate enforcement |

### 6.2 Most Dangerous Collapse

```
Derived provenance silently mutating into authority provenance
```

This is the repository's greatest remaining constitutional risk.

**Mechanism:**
1. Value derived with PROV_DERIVATION
2. Value cached without provenance
3. Cached value consumed
4. Consumer treats as authoritative
5. Authority provenance implicitly assumed

**Prevention:**
- Provenance must propagate through caching
- Authority-state must persist
- Consumers must honor provenance type
