# Features Overview

Luthier's ToolBox provides a comprehensive suite of tools for instrument design and CNC manufacturing.

---

## Calculator Tools

### Scale Length Designer

Calculate string tension using Mersenne's Law. Compare different scale lengths, string gauges, and tunings.

**Key Features:**

- Multi-scale (fanned fret) support
- Tension comparison charts
- Preset library (Fender, Gibson, PRS, etc.)
- Custom tuning support

[Learn More →](scale-length.md)

---

### Unit Converter

Convert between all units common in lutherie and woodworking.

**Categories:**

- Length (mm, inches, thou)
- Mass (grams, ounces, pounds)
- Temperature (°C, °F, K)
- Board Feet calculations
- Data units (bytes, KB, MB, GB)

[Learn More →](unit-converter.md)

---

### Woodwork Calculator

Calculate material requirements and angles.

**Features:**

- Board feet calculator
- Wood weight estimation by species
- Miter angle calculator
- Compound angle calculator

[Learn More →](woodwork-calculator.md)

---

### Fret Calculator

Calculate fret positions for any scale length and temperament.

**Features:**

- Equal temperament (12-TET) default
- Custom ratio support for historical temperaments
- Compensation calculations
- Export to DXF/CSV

[Learn More →](fret-calculator.md)

---

## CAD/CAM Tools

### DXF Import

Import and validate CAD files for CNC processing.

**Features:**

- Geometry validation
- Open contour detection
- Self-intersection repair
- Layer management

[Learn More →](dxf-import.md)

---

### Toolpath Generation

Generate G-code for various CNC operations.

**Operations:**

- Adaptive pocket clearing
- Profile/contour cutting
- Drilling and boring
- V-carving

[Learn More →](toolpaths.md)

---

### Rosette Designer

Create decorative sound hole patterns.

**Patterns:**

- Concentric rings
- Radial segments
- Tessellated tiles
- Custom SVG import

[Learn More →](rosette-designer.md)

---

## Safety & Intelligence

### RMOS (Run Management & Operations System)

Decision intelligence for manufacturing safety.

**Features:**

- Feasibility analysis (GREEN/YELLOW/RED)
- Rule-based safety checks
- Operator override with audit trail
- Validation protocol

[Learn More →](../cam/safety-rmos.md)
